
# AI Resume Analyzer

AI Resume Analyzer is a web application that compares a resume with a job description using NLP techniques and calculates a matching percentage.

---

## Features

- Upload Resume PDF
- Compare Resume with Job Description
- Skill Extraction
- Match Percentage
- Missing Skills Detection

---

## Technologies Used

- Python
- Flask
- Scikit-learn
- PDFPlumber
- HTML
- CSS

---


## Installation

### Clone Repository

git clone YOUR_REPOSITORY_LINK

### Open Project Folder

cd AI-Resume-Analyzer

### Install Required Libraries

pip install -r requirements.txt

### Run Application

python app.py

---

## Output

The application displays:

- Resume Match Percentage
- Skills Found
- Missing Skills
